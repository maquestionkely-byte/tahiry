  <!-- Carte : Inona ny double pronom ? -->
  <div class="card">
    <h2>🏷️ Inona ny "double pronom" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>double pronom</strong> dia rehefa misy <strong>pronom roa</strong> ao anaty fehezanteny iray : ny <strong>COD</strong> (Complément d'Objet Direct) sy ny <strong>COI</strong> (Complément d'Objet Indirect).<br><br>
      🔹 Mitovy amin'ny hoe <strong>"...azy"</strong> sy <strong>"...azy"</strong> amin'ny teny malagasy.<br><br>
      🔹 Misy <strong>filaharana</strong> (ordre) tsy maintsy arahina rehefa mampiasa pronom roa.<br><br>
      <span class="icone">📌</span> <strong>Ordre général :</strong><br>
      <span class="color-sujet">Sujet</span> + <span class="color-me-te-se-nous-vous">me / te / se / nous / vous</span> + <span class="color-le-la-les">le / la / les</span> + <span class="color-lui-leur">lui / leur</span> + <span class="color-y">y</span> + <span class="color-en">en</span> + <span class="color-verbe">verbe</span><br><br>
      🔹 Ohatra :<br>
      • <em><span class="color-sujet">Il</span> <span class="color-le-la-les">le</span> <span class="color-lui-leur">lui</span> <span class="color-verbe">donne</span>.</em> (Omeny azy izy.)<br>
      • <em><span class="color-sujet">Je</span> <span class="color-me-te-se-nous-vous">te</span> <span class="color-le-la-les">le</span> <span class="color-verbe">donne</span>.</em> (Omeo ahy izy.)

  <div class="card">
    <h2>1️⃣ <span class="color-me-te-se-nous-vous">me / te / se / nous / vous</span> + <span class="color-le-la-les">le / la / les</span></h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Rehefa misy <span class="color-me-te-se-nous-vous">me, te, se, nous, vous</span> sy <span class="color-le-la-les">le, la, les</span> miaraka, dia apetraka <strong>aloha</strong> ny <span class="color-me-te-se-nous-vous">me, te, se, nous, vous</span>.<br><br>
      <strong>Structure :</strong><br>
      <em><span class="color-me-te-se-nous-vous">me/te/se/nous/vous</span> + <span class="color-le-la-les">le/la/les</span> + verbe</em>

      <div class="exo-ligne">
        <span class="fr">Il <span class="color-me-te-se-nous-vous">se</span> <span class="color-le-la-les">le</span> dit. (Lazainy amin'ny tenany izy.)</span>
      </div>
      <div class="exo-ligne">

      </div>
      <div class="exo-ligne">
