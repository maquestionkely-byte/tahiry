  <!-- Carte : Inona ny accord du participe passé ? -->
  <div class="card">
    <h2>🏷️ Inona ny "accord du participe passé" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>accord du participe passé</strong> dia ny <strong>fifanarahana</strong> eo amin'ny <strong>participe passé</strong> sy ny <strong>sujet</strong> na ny <strong>COD</strong> (Complément d'Objet Direct).<br><br>
      🔹 Miankina amin'ny <strong>auxiliaire</strong> ampiasaina (<span class="color-avoir">avoir</span> ou <span class="color-etre">être</span>) ny fomba fifanarahana.<br><br>
      🔹 Misy fitsipika (règles) telo lehibe :<br><br>
      <span class="icone">📌</span> <strong>Avec "être"</strong> → le participe passé s'accorde <strong>en genre et en nombre</strong> avec le <strong>sujet</strong>.<br>
      Ohatra : <em>Elle est <span class="color-etre">partie</span>.</em> (Efa lasa izy.)<br><br>
      <span class="icone">📌</span> <strong>Avec "avoir"</strong> → le participe passé s'accorde <strong>en genre et en nombre</strong> avec le <strong>COD</strong> si celui-ci est <strong>placé avant</strong> le verbe.<br>
      Ohatra : <em>Les pommes que j'ai <span class="color-avoir">mangées</span>.</em> (Ireo paoma izay nohaniko.)<br><br>
      <span class="icone">📌</span> <strong>Verbes pronominaux</strong> → règles spécifiques.<br>
      Ohatra : <em>Elle s'est <span class="color-pronominal">lavée</span>.</em> (Efa nanasa tena izy.)
    </div>
  </div>

  <!-- ==================== TABLEAU RÉCAPITULATIF ==================== -->
  <div class="card">
    <h2>📊 Tableau récapitulatif (Famintinana)</h2>
    <div class="tableau-recap">
      <table>
        <thead>
          <tr>
            <th>Auxiliaire</th>
            <th>Règle d'accord</th>
            <th>Exemple</th>
          </tr>
        </thead>
        <tbody>
          <tr class="categorie">
            <td><span class="color-etre">ÊTRE</span></td>
            <td>S'accorde avec le <strong>sujet</strong> (genre + nombre)</td>
            <td>Elle est <strong>partie</strong>.</td>
          </tr>
          <tr class="categorie">
            <td><span class="color-avoir">AVOIR</span></td>
            <td>S'accorde avec le <strong>COD</strong> si placé <strong>avant</strong></td>
            <td>Les pommes que j'ai <strong>mangées</strong>.</td>
          </tr>
          <tr class="categorie">
            <td><span class="color-pronominal">PRONOMINAL</span></td>
            <td>S'accorde avec le <strong>sujet</strong> si le pronom est COD</td>
            <td>Elle s'est <strong>lavée</strong>.</td>
          </tr>
        </tbody>
      </table>
    </div>
    
    <div class="important-remark">
      <strong>📌 RÈGLES IMPORTANTES :</strong><br><br>
      • <span class="color-etre">Avec "être"</span> → le participe passé s'accorde <strong>toujours</strong> avec le sujet.<br>
      Ohatra : <em>Marie est <strong>allée</strong>.</em> / <em>Ils sont <strong>allés</strong>.</em><br><br>
      • <span class="color-avoir">Avec "avoir"</span> → le participe passé <strong>ne s'accorde pas</strong> avec le sujet, mais avec le <strong>COD</strong> si celui-ci est <strong>avant</strong> le verbe.<br>
      Ohatra : <em>J'ai <strong>mangé</strong> des pommes.</em> (COD après, pas d'accord)<br>
      Ohatra : <em>Les pommes que j'ai <strong>mangées</strong>.</em> (COD avant, accord)<br><br>
      • <span class="color-pronominal">Verbes pronominaux</span> → l'accord dépend de la place du pronom réfléchi (COD ou COI).<br>
      Ohatra : <em>Elle s'est <strong>lavée</strong>.</em> (se = COD, accord)<br>
      Ohatra : <em>Elles se sont <strong>parlé</strong>.</em> (se = COI, pas d'accord)
    </div>
  </div>

  <!-- 1. ACCORD AVEC "ÊTRE" -->
  <div class="card">
    <h2>1️⃣ Accord avec <span class="color-etre">ÊTRE</span></h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Rehefa mampiasa ny auxiliaire <span class="color-etre">être</span>, ny <span class="color-participe">participe passé</span> dia <strong>mifanaraka</strong> amin'ny <strong>sujet</strong> (genre et nombre).<br><br>
      <strong>Règle :</strong><br>
      • Masculin singulier → tsy misy fanampiny (allé)<br>
      • Féminin singulier → manampy <strong>"e"</strong> (allée)<br>
      • Masculin pluriel → manampy <strong>"s"</strong> (allés)<br>
      • Féminin pluriel → manampy <strong>"es"</strong> (allées)
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Il est <span class="color-participe">allé</span>. (Masculin singulier.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elle est <span class="color-participe">allée</span>. (Féminin singulier.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Ils sont <span class="color-participe">allés</span>. (Masculin pluriel.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elles sont <span class="color-participe">allées</span>. (Féminin pluriel.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Nous sommes <span class="color-participe">allés</span> au cinéma. (Pluriel - masculin l'emporte si groupes mixtes.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Marie est <span class="color-participe">venue</span> avec nous. (Féminin singulier.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Rehefa mampiasa <span class="color-etre">être</span>, ny participe passé dia <strong>mifanaraka</strong> amin'ny sujet.
    </div>
  </div>

  <!-- 2. ACCORD AVEC "AVOIR" -->
  <div class="card">
    <h2>2️⃣ Accord avec <span class="color-avoir">AVOIR</span></h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Rehefa mampiasa ny auxiliaire <span class="color-avoir">avoir</span>, ny <span class="color-participe">participe passé</span> dia <strong>tsy</strong> mifanaraka amin'ny sujet, fa <strong>mifanaraka</strong> amin'ny <strong>COD</strong> (Complément d'Objet Direct) raha <strong>eo alohan'ny</strong> verbe izy.<br><br>
      <strong>Règle :</strong><br>
      • Raha <strong>tsy misy COD</strong> na misy COD <strong>aorian'ny</strong> verbe → tsy misy accord.<br>
      • Raha misy COD <strong>alohan'ny</strong> verbe → mifanaraka amin'ny COD (genre + nombre).
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay (Pas d'accord)
      </div>
      <div class="exo-ligne">
        <span class="fr">J'ai <span class="color-participe">mangé</span> une pomme. (COD après → pas d'accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il a <span class="color-participe">fini</span> ses devoirs. (COD après → pas d'accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Nous avons <span class="color-participe">chanté</span> une chanson. (COD après → pas d'accord.)</span>
      </div>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay (Avec accord)
      </div>
      <div class="exo-ligne">
        <span class="fr">La pomme que j'ai <span class="color-participe">mangée</span>. (COD "la pomme" avant → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Les devoirs qu'il a <span class="color-participe">finis</span>. (COD "les devoirs" avant → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Les chansons que nous avons <span class="color-participe">chantées</span>. (COD "les chansons" avant → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Je les ai <span class="color-participe">vues</span>. (COD "les" (féminin pluriel) avant → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il les a <span class="color-participe">invités</span>. (COD "les" (masculin pluriel) avant → accord.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Rehefa mampiasa <span class="color-avoir">avoir</span>, ny participe passé dia <strong>tsy mifanaraka</strong> amin'ny sujet, fa miaraka amin'ny <strong>COD alohan'ny</strong> verbe.
    </div>
  </div>

  <!-- 3. ACCORD AVEC LES VERBES PRONOMINAUX -->
  <div class="card">
    <h2>3️⃣ Accord avec les verbes pronominaux</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny verbes pronominaux (se laver, se parler, etc.) dia mampiasa ny auxiliaire <span class="color-etre">être</span>. Ny participe passé dia <strong>mifanaraka</strong> amin'ny <strong>sujet</strong> raha ny pronom réfléchi (se) no <strong>COD</strong>.<br><br>
      <strong>Règle :</strong><br>
      • Raha <span class="color-pronominal">se</span> = COD → mifanaraka amin'ny sujet.<br>
      • Raha <span class="color-pronominal">se</span> = COI (Complément d'Objet Indirect) → tsy mifanaraka.
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay (se = COD → accord)
      </div>
      <div class="exo-ligne">
        <span class="fr">Elle s'est <span class="color-participe">lavée</span>. (se = COD (elle a lavé qui? → elle-même) → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Ils se sont <span class="color-participe">réveillés</span> tôt. (se = COD → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elles se sont <span class="color-participe">habillées</span> rapidement. (se = COD → accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Je me suis <span class="color-participe">brossé</span> les dents. (se = COI (brossé quoi? → les dents) → pas d'accord.)</span>
      </div>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay (se = COI → pas d'accord)
      </div>
      <div class="exo-ligne">
        <span class="fr">Elles se sont <span class="color-participe">parlé</span>. (se = COI (parlé à qui? → à elles) → pas d'accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Ils se sont <span class="color-participe">écrit</span> des lettres. (se = COI (écrit à qui? → à eux) → pas d'accord.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Nous nous sommes <span class="color-participe">plu</span>. (se = COI (plu à qui? → à nous) → pas d'accord.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Amin'ny verbes pronominaux, ny participe passé dia <strong>mifanaraka</strong> amin'ny sujet raha ny pronom réfléchi (se) dia <strong>COD</strong>. Raha <strong>COI</strong> izy, dia <strong>tsy</strong> mifanaraka.
    </div>
  </div>

  <!-- 4. LISTE DES VERBES PRONOMINAUX AVEC SE = COD -->

  <!-- 5. LISTE DES VERBES PRONOMINAUX AVEC SE = COI -->
  <div class="card">
    <h2>5️⃣ Verbes pronominaux avec <span class="color-pronominal">se = COI</span> (pas d'accord)</h2>
    <div class="definition-mg">

  <div class="card">
    <h2>📌 Résumé des règles d'accord</h2>
    <div class="definition-mg">
      <strong>1. Avec "être" :</strong><br>
      • Le participe passé s'accorde <strong>toujours</strong> avec le <strong>sujet</strong>.<br>
      Ohatra : <em>Elle est <span class="color-etre">partie</span>.</em> / <em>Ils sont <span class="color-etre">partis</span>.</em><br><br>
      
      <strong>2. Avec "avoir" :</strong><br>
      • Le participe passé <strong>ne s'accorde pas</strong> avec le sujet.<br>
      • Le participe passé s'accorde avec le <strong>COD</strong> si celui-ci est <strong>placé avant</strong> le verbe.<br>
      Ohatra : <em>J'ai <span class="color-avoir">mangé</span> des pommes.</em> (pas d'accord)<br>
      Ohatra : <em>Les pommes que j'ai <span class="color-avoir">mangées</span>.</em> (accord)<br><br>
      
      <strong>3. Avec les verbes pronominaux :</strong><br>
      • Si <span class="color-pronominal">se</span> = <strong>COD</strong> → accord avec le sujet.<br>
      Ohatra : <em>Elle s'est <span class="color-pronominal">lavée</span>.</em><br>
      • Si <span class="color-pronominal">se</span> = <strong>COI</strong> → pas d'accord.<br>
      Ohatra : <em>Elles se sont <span class="color-pronominal">parlé</span>.</em>
    </div>
  </div>

    <h2>🎯 Torohevitra mahasoa ho an'ny débutant</h2>
    <div class="definition-mg" style="margin-bottom: 6px;">
      <ul class="liste-avec-lignes-vides">
        <li>Ny <strong>accord du participe passé</strong> dia miankina amin'ny <strong>auxiliaire</strong> ampiasaina.</li>
        <li>Rehefa mampiasa <span class="color-etre">être</span> → <strong>mifanaraka</strong> amin'ny sujet.</li>
        <li>Rehefa mampiasa <span class="color-avoir">avoir</span> → <strong>tsy</strong> mifanaraka amin'ny sujet, fa miaraka amin'ny <strong>COD alohan'ny</strong> verbe.</li>
        <li>Raha <strong>tsy misy COD</strong> na misy COD <strong>aorian'ny</strong> verbe → <strong>tsy</strong> misy accord.</li>
        <li>Raha misy COD <strong>alohan'ny</strong> verbe → <strong>mifanaraka</strong> amin'ny COD.</li>
        <li>Amin'ny verbes pronominaux → jereo raha <span class="color-pronominal">se</span> = <strong>COD</strong> (accord) na <strong>COI</strong> (pas d'accord).</li>
        <li>Vakio mafy ny ohatra isan'andro mba hahatsapanao ny fahasamihafana.</li>
      </ul>
    </div>
