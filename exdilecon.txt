  <div class="card">
    <h2>🏷️ Inona ny "exprimer le but" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>but</strong> dia mamaritra ny <strong>tanjona</strong> (objectif) na ny <strong>antony</strong> (raison) anaovana zavatra iray.<br><br>
      🔹 Mitovy amin'ny hoe <strong>"mba", "ho an'ny", "amin'ny tanjona"</strong> amin'ny teny malagasy.<br><br>
      🔹 Misy fomba <strong>maro</strong> hilazana ny tanjona :<br><br>
      <span class="icone">📌</span> <span class="color-pour">POUR</span> → mba (tanjona tsotra)<br>
      Ohatra : <em>Il étudie <span class="color-pour">pour</span> réussir.</em><br><br>
      <span class="icone">📌</span> <span class="color-afin-de">AFIN DE</span> → mba (tanjona ara-dalàna)<br>
      Ohatra : <em>Il étudie <span class="color-afin-de">afin de</span> réussir.</em><br><br>
      <span class="icone">📌</span> <span class="color-dans-le-but-de">DANS LE BUT DE</span> → amin'ny tanjona (ara-dalàna)<br>
      Ohatra : <em>Il étudie <span class="color-dans-le-but-de">dans le but de</span> réussir.</em><br><br>
      <span class="icone">📌</span> <span class="color-pour-que">POUR QUE</span> → mba (tanjona + subjonctif)<br>
      Ohatra : <em>Il étudie <span class="color-pour-que">pour qu'</span>il réussisse.</em><br><br>
      <span class="icone">📌</span> <span class="color-afin-que">AFIN QUE</span> → mba (tanjona + subjonctif - ara-dalàna)<br>
      Ohatra : <em>Il étudie <span class="color-afin-que">afin qu'</span>il réussisse.</em>
    </div>

  <div class="card">
    <h2>1️⃣ <span class="color-pour">POUR</span> (mba - tanjona tsotra)</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-pour">POUR</span> dia ampiasaina hilazana <strong>tanjona</strong> tsotra. Arahin'ny <strong>infinitif</strong>.<br><br>
      <strong>Règle :</strong><br>
      • <span class="color-pour">POUR</span> + <strong>infinitif</strong><br>
      • Mamaly ny fanontaniana <strong>"Pourquoi ?"</strong> (Nahoana? / Ho an'inona?)<br>
      • Ampiasaina amin'ny resaka andavanandro
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Il étudie <span class="color-pour">pour</span> réussir. (Mianatra izy <span class="color-pour">mba</span> hahomby.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elle va à la salle de sport <span class="color-pour">pour</span> se muscler. (Mandeha any amin'ny toeram-pilalaovana izy <span class="color-pour">mba</span> hanamafy vatana.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il travaille <span class="color-pour">pour</span> gagner de l'argent. (Miasa izy <span class="color-pour">mba</span> hahazo vola.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Je suis venu <span class="color-pour">pour</span> te voir. (Tonga aho <span class="color-pour">mba</span> hahita anao.)</span>
      </div>
    </div>
    <div class="highlight">

      • <span class="color-afin-que">AFIN QUE</span> + <strong>sujet + verbe (subjonctif)</strong><br>

  <div class="card">
    <h2>6️⃣ Comparaison POUR QUE vs AFIN QUE</h2>
    <div class="definition-mg">
      <strong><span class="color-pour-que">POUR QUE</span></strong> → mba (tanjona + subjonctif)<br>
      Ohatra : <em>Il étudie <span class="color-pour-que">pour qu'</span>il réussisse.</em><br><br>
      
      <strong><span class="color-afin-que">AFIN QUE</span></strong> → mba (tanjona + subjonctif - ara-dalàna)<br>
      Ohatra : <em>Il étudie <span class="color-afin-que">afin qu'</span>il réussisse.</em><br><br>
      
      <strong>Fahasamihafana :</strong><br>
      • <span class="color-pour-que">POUR QUE</span> → ampiasaina amin'ny resaka <strong>andavanandro</strong> (courant).<br>
      • <span class="color-afin-que">AFIN QUE</span> → ampiasaina amin'ny fomba <strong>ara-dalàna</strong> (formel).<br>
      • Samy arahin'ny <strong>subjonctif</strong> izy roa.<br>
      • Samy ampiasaina rehefa <strong>tsy mitovy</strong> ny sujet.
    </div>

  <div class="card">
    <h2>📌 Résumé des règles</h2>
    <div class="definition-mg">
      <strong>1. <span class="color-pour">POUR</span></strong> → mba (tanjona tsotra + infinitif)<br>
      Ohatra : <em>Il étudie pour réussir.</em><br><br>
      
      <strong>2. <span class="color-afin-de">AFIN DE</span></strong> → mba (tanjona ara-dalàna + infinitif)<br>
      Ohatra : <em>Il étudie afin de réussir.</em><br><br>
      
      <strong>3. <span class="color-dans-le-but-de">DANS LE BUT DE</span></strong> → amin'ny tanjona (très formel + infinitif)<br>
      Ohatra : <em>Il étudie dans le but de réussir.</em><br><br>
      
      <strong>4. <span class="color-pour-que">POUR QUE</span></strong> → mba (tanjona + subjonctif - andavanandro)<br>
      Ohatra : <em>Il étudie pour qu'il réussisse.</em><br><br>
      
      <strong>5. <span class="color-afin-que">AFIN QUE</span></strong> → mba (tanjona + subjonctif - ara-dalàna)<br>
      Ohatra : <em>Il étudie afin qu'il réussisse.</em>
