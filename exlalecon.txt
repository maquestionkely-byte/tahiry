  <div class="card">
    <h2>🏷️ Inona ny "exprimer la cause" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>cause</strong> dia mamaritra ny <strong>antony</strong> (raison) nahatonga zavatra iray hitranga.<br><br>
      🔹 Mitovy amin'ny hoe <strong>"satria", "noho", "amin'ny alalan'ny"</strong> amin'ny teny malagasy.<br><br>
      🔹 Misy fomba <strong>maro</strong> hilazana ny antony :<br><br>
      <span class="icone">📌</span> <span class="color-parce-que">PARCE QUE</span> → satria (antony tsotra)<br>
      Ohatra : <em>Il dort <span class="color-parce-que">parce qu'</span>il est fatigué.</em><br><br>
      <span class="icone">📌</span> <span class="color-car">CAR</span> → satria (ara-dalàna kokoa)<br>
      Ohatra : <em>Il dort, <span class="color-car">car</span> il est fatigué.</em><br><br>
      <span class="icone">📌</span> <span class="color-puisque">PUISQUE</span> → satria (efa fantatra ny antony)<br>
      Ohatra : <em><span class="color-puisque">Puisque</span> tu es là, reste.</em><br><br>
      <span class="icone">📌</span> <span class="color-a-cause-de">À CAUSE DE</span> → noho ny (antony ratsy)<br>
      Ohatra : <em>Il est en retard <span class="color-a-cause-de">à cause de</span> la pluie.</em><br><br>
      <span class="icone">📌</span> <span class="color-grace-a">GRÂCE À</span> → noho ny (antony tsara)<br>
      Ohatra : <em>Il a réussi <span class="color-grace-a">grâce à</span> son travail.</em>
    </div>

    <h2>1️⃣ <span class="color-parce-que">PARCE QUE</span> (satria - antony tsotra)</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-parce-que">PARCE QUE</span> dia ampiasaina hilazana <strong>antony tsotra</strong>. Mamaly ny fanontaniana <strong>"Pourquoi ?"</strong> (Nahoana?).<br><br>
      <strong>Règle :</strong><br>
      • <span class="color-parce-que">PARCE QUE</span> + <strong>sujet + verbe</strong> (proposition subordonnée)<br>
      • Apetraka <strong>aorian'ny</strong> proposition principale<br>
      • Miova ho <span class="color-parce-que">PARCE QU'</span> alohan'ny zanatsoratra
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Il dort <span class="color-parce-que">parce qu'</span>il est fatigué. (Matory izy <span class="color-parce-que">satria</span> vizana.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elle pleure <span class="color-parce-que">parce qu'</span>elle est triste. (Mitomany izy <span class="color-parce-que">satria</span> malahelo.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Je reste à la maison <span class="color-parce-que">parce qu'</span>il pleut. (Mijanona ao an-trano aho <span class="color-parce-que">satria</span> misy orana.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il est content <span class="color-parce-que">parce qu'</span>il a réussi. (Faly izy <span class="color-parce-que">satria</span> nahomby.)</span>

  <!-- 6. COMPARAISON À CAUSE DE vs GRÂCE À -->
  <div class="card">
    <h2>6️⃣ Comparaison À CAUSE DE vs GRÂCE À</h2>
    <div class="definition-mg">
      <strong><span class="color-a-cause-de">À CAUSE DE</span></strong> → noho ny (antony <strong>ratsy</strong>)<br>
      Ohatra : <em>Il est en retard <span class="color-a-cause-de">à cause de</span> la pluie.</em> (Tara izy <span class="color-a-cause-de">noho ny</span> orana.)<br><br>
      
      <strong><span class="color-grace-a">GRÂCE À</span></strong> → noho ny (antony <strong>tsara</strong>)<br>

      • <span class="color-a-cause-de">À CAUSE DE</span> → ampiasaina amin'ny zavatra <strong>ratsy</strong>.<br>
      • <span class="color-grace-a">GRÂCE À</span> → ampiasaina amin'ny zavatra <strong>tsara</strong>.<br>

  <div class="card">
    <h2>📌 Résumé des règles</h2>
    <div class="definition-mg">
      <strong>1. <span class="color-parce-que">PARCE QUE</span></strong> → satria (antony tsotra)<br>
      • Structure : <em>Proposition principale + <span class="color-parce-que">parce que</span> + sujet + verbe</em><br>
      Ohatra : <em>Il dort parce qu'il est fatigué.</em><br><br>
      
      <strong>2. <span class="color-car">CAR</span></strong> → satria (ara-dalàna)<br>
      • Structure : <em>Proposition principale, <span class="color-car">car</span> + sujet + verbe</em><br>
      Ohatra : <em>Il dort, car il est fatigué.</em><br><br>
      
      <strong>3. <span class="color-puisque">PUISQUE</span></strong> → satria (antony efa fantatra)<br>
      • Structure : <em><span class="color-puisque">Puisque</span> + sujet + verbe, proposition principale</em><br>
      Ohatra : <em>Puisque tu es là, reste.</em><br><br>
      
      <strong>4. <span class="color-a-cause-de">À CAUSE DE</span></strong> → noho ny (antony ratsy)<br>
      • Structure : <em>Proposition principale + <span class="color-a-cause-de">à cause de</span> + nom</em><br>
      Ohatra : <em>Il est en retard à cause de la pluie.</em><br><br>
      
      <strong>5. <span class="color-grace-a">GRÂCE À</span></strong> → noho ny (antony tsara)<br>
      • Structure : <em>Proposition principale + <span class="color-grace-a">grâce à</span> + nom</em><br>
      Ohatra : <em>Il a réussi grâce à son travail.</em>
    </div>
