    <div class="definition-mg">
      🔹 Ny <strong>phrase simple</strong> dia fehezanteny <strong>tsotra</strong> izay misy <span class="color-sujet">sujet</span> (mpanao), <span class="color-verbe">verbe</span> (mpamaritra), ary <span class="color-complement">complément</span> (fanampiny) indraindray.<br><br>
      🔹 Misy fitsipika (règles) fototra tokony hotadidina :<br><br>
      <span class="icone">📌</span> <strong>Structure de base :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span> + <span class="color-complement">Complément</span><br>
      Ohatra : <em><span class="color-sujet">Le chat</span> <span class="color-verbe">mange</span> <span class="color-complement">une pomme</span>.</em> (Mihinana paoma ny saka.)<br><br>
      <span class="icone">📌</span> <strong>Phrase minimale :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span><br>
      Ohatra : <em><span class="color-sujet">Il</span> <span class="color-verbe">dort</span>.</em> (Matoky izy.)<br><br>
      <span class="icone">📌</span> <strong>Phrase avec complément :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span> + <span class="color-complement">Complément</span><br>
      Ohatra : <em><span class="color-sujet">Marie</span> <span class="color-verbe">lit</span> <span class="color-complement">un livre</span>.</em> (Mamaky boky i Marie.)

  <!-- 1. SUJET - Le sujet -->
  <div class="card">
    <h2>1️⃣ <span class="color-sujet">SUJET</span> (mpanao)</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-sujet">sujet</span> dia ilay <strong>mpanao</strong> zavatra. Izy no <strong>lohahevitra</strong> (thème) voalaza ao amin'ny fehezanteny.<br><br>
      <strong>Règle :</strong><br>
      • Matetika dia <strong>eo alohan'ny</strong> verbe ny sujet.<br>
      • Mety ho <strong>pronom</strong> (je, tu, il...) na <strong>nom</strong> (le chat, Marie).<br>
      • Ny sujet dia <strong>miara-mifanaraka</strong> amin'ny verbe (singulier ou pluriel).
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr"><span class="color-sujet">Le chat</span> dort. (Matoky ny <span class="color-sujet">saka</span>.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr"><span class="color-sujet">Marie</span> lit un livre. (Mamaky boky i <span class="color-sujet">Marie</span>.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr"><span class="color-sujet">Il</span> mange une pomme. (Mihinana paoma <span class="color-sujet">izy</span>.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr"><span class="color-sujet">Nous</span> allons à l'école. (Mandeha any an-tsekoly <span class="color-sujet">izahay</span>.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Ny <span class="color-sujet">sujet</span> dia ilay <strong>mpanao</strong> zavatra. Matetika izy no <strong>mialoha</strong> ny verbe.
    </div>
  </div>

    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-verbe">verbe</span> dia ilay <strong>mpamaritra</strong> zavatra ataon'ny sujet. Izy no <strong>mpitantana</strong> ny fehezanteny.<br><br>
      <strong>Règle :</strong><br>
      • Matetika dia <strong>eo aorian'ny</strong> sujet ny verbe.<br>
      • Ny verbe dia <strong>mifanaraka</strong> amin'ny sujet (singulier ou pluriel).<br>
      • Misy karazany roa : <strong>verbe d'état</strong> (être, sembler...) sy <strong>verbe d'action</strong> (manger, courir...).
    </div>

    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>

    </div>
  </div>

  <!-- ==================== LA PHRASE SIMPLE EN DÉTAIL ==================== -->
  <div class="card">
    <h2>📌 La phrase simple en détail</h2>
    <div class="definition-mg">
      <strong>1. Phrase minimale :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span><br>
      Ohatra : <em><span class="color-sujet">Il</span> <span class="color-verbe">dort</span>.</em><br><br>
      
      <strong>2. Phrase avec complément d'objet :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span> + <span class="color-complement">COD</span><br>
      Ohatra : <em><span class="color-sujet">Il</span> <span class="color-verbe">mange</span> <span class="color-complement">une pomme</span>.</em><br><br>
      
      <strong>3. Phrase avec complément circonstanciel :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span> + <span class="color-complement">C.C.</span><br>
      Ohatra : <em><span class="color-sujet">Il</span> <span class="color-verbe">court</span> <span class="color-complement">vite</span>.</em><br><br>
      
      <strong>4. Phrase avec deux compléments :</strong> <span class="color-sujet">Sujet</span> + <span class="color-verbe">Verbe</span> + <span class="color-complement">COD</span> + <span class="color-complement">C.C.</span><br>
      Ohatra : <em><span class="color-sujet">Il</span> <span class="color-verbe">mange</span> <span class="color-complement">une pomme</span> <span class="color-complement">le matin</span>.</em>
    </div>
    <div class="exemple-modern" style="margin-top: 12px;">
      <div class="exo-titre">
        <span>🧪</span> Ohatra fampitahana mazava
      </div>
      <div class="exo-ligne">
        <span class="fr">MINIMALE : <span class="color-sujet">Il</span> <span class="color-verbe">dort</span>.</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">AVEC COD : <span class="color-sujet">Il</span> <span class="color-verbe">mange</span> <span class="color-complement">une pomme</span>.</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">AVEC C.C. : <span class="color-sujet">Il</span> <span class="color-verbe">court</span> <span class="color-complement">vite</span>.</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">AVEC DEUX : <span class="color-sujet">Il</span> <span class="color-verbe">mange</span> <span class="color-complement">une pomme</span> <span class="color-complement">le matin</span>.</span>
      </div>
    </div>
  </div>

  <!-- ==================== LES TYPES DE COMPLÉMENTS ==================== -->
  <div class="card">
    <h2>📌 Les types de compléments</h2>
    <div class="definition-mg">
      <strong>1. Le COD (Complément d'Objet Direct) :</strong><br>
      • Mamaritra <span class="color-complement">inona</span> na <span class="color-complement">iza</span> ny zavatra atao.<br>
      • Tsy misy préposition (à, de, pour) eo alohany.<br>
      Ohatra : <em>Il mange <span class="color-complement">une pomme</span>.</em> (Inona no hohaniny? = une pomme)<br><br>
      
      <strong>2. Le COI (Complément d'Objet Indirect) :</strong><br>
      • Mamaritra <span class="color-complement">amin'iza</span> na <span class="color-complement">ho an'iza</span> ny zavatra atao.<br>
      • Misy préposition (à, de) eo alohany.<br>
      Ohatra : <em>Il parle <span class="color-complement">à son ami</span>.</em> (Amin'iza no itenenany? = à son ami)<br><br>
      
      <strong>3. Le Complément Circonstanciel (C.C.) :</strong><br>
      • Mamaritra ny <span class="color-complement">fotoana</span>, <span class="color-complement">toerana</span>, <span class="color-complement">fomba</span>, <span class="color-complement">habetsahana</span>...<br>
      Ohatra : <em>Il court <span class="color-complement">vite</span>.</em> (Ahoana ny fihazakazahany? = vite)
    </div>
    <div class="exemple-modern" style="margin-top: 12px;">
      <div class="exo-titre">
        <span>🧪</span> Ohatra amin'ny teny français

      <strong>Règle générale :</strong><br>
      Amin'ny teny frantsay, ny filaharan-teny mahazatra dia :<br><br>
      <span class="color-sujet">SUJET</span> + <span class="color-verbe">VERBE</span> + <span class="color-complement">COMPLÉMENT</span><br><br>
