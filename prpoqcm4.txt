Exercice : Pronoms possessifs - Niveau 4 (difficile) - Notre/nôtre, votre/vôtre

Consigne : Choisissez le bon pronom possessif.

1. Ce livre est à nous, c'est ___.
A) le nôtre
B) la nôtre
C) les nôtres
D) le vôtre
*A) le nôtre
TRADUCTION: Ahy ity boky ity.

2. Cette maison est à vous, c'est ___.
A) la nôtre
B) la vôtre
C) la leur
D) la mienne
*B) la vôtre
TRADUCTION: Anareo ity trano ity.

3. Ces idées sont à nous, ce sont ___.
A) le nôtre
B) la nôtre
C) les nôtres
D) les vôtres
*C) les nôtres
TRADUCTION: Ahy ireo hevitra ireo.

4. Ces chaussures sont à vous, ce sont ___.
A) le nôtre
B) la nôtre
C) les nôtres
D) les vôtres
*D) les vôtres
TRADUCTION: Anareo ireo kiraro ireo.

5. Ton stylo est noir, ___ est bleu.
A) le nôtre
B) le vôtre
C) le leur
D) le mien
*A) le nôtre
TRADUCTION: Mainty ny peninao, manga ny anay.

6. Ta voiture est rapide, ___ est lente.
A) la nôtre
B) la vôtre
C) la leur
D) la tienne
*B) la vôtre
