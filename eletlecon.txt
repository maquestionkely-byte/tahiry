  <!-- 1. L'ELLIPSE - Éviter les répétitions -->
  <div class="card">
    <h2>1️⃣ <span class="color-ellipse">ELLIPSE</span> (mampihena ny fehezanteny)</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-ellipse">ellipse</span> dia fomba <strong>manafoana</strong> (mampihena) ny fehezanteny amin'ny alalan'ny fanesorana teny tsy ilaina, nefa tsy miova ny heviny.<br><br>
      <strong>Règle :</strong><br>
      • Azo esorina ny <strong>verbe</strong> raha miverina mitovy.<br>
      • Azo esorina ny <strong>sujet</strong> raha mazava.<br>
      • Azo esorina ny <strong>complément</strong> raha tsy ilaina.<br>
      Ohatra : <em>Pierre aime le café, Marie aime le café → Pierre aime le café, Marie <span class="color-ellipse">aussi</span>.</em>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Pierre aime le café, Marie <span class="color-ellipse">aussi</span>. (Tia kafe i Pierre, <span class="color-ellipse">tia</span> koa i Marie.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Je mange une pomme, <span class="color-ellipse">toi</span> une banane. (Mihinana paoma aho, <span class="color-ellipse">mihinana</span> akondro ianao.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il parle français, <span class="color-ellipse">elle</span> anglais. (Miteny frantsay izy, <span class="color-ellipse">miteny</span> anglisy izy.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Paul est grand, <span class="color-ellipse">Marie</span> petite. (Lehibe i Paul, <span class="color-ellipse">kely</span> i Marie.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Ny <span class="color-ellipse">ellipse</span> dia ampiasaina <strong>hampihenana</strong> ny fehezanteny nefa tsy miova ny heviny.
    </div>
  </div>

  <!-- 2. L'ANAPHORE - Reprendre ce qui a été dit -->
  <div class="card">
    <h2>2️⃣ <span class="color-anaphore">ANAPHORE</span> (manondro zavatra efa voalaza)</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-anaphore">anaphore</span> dia fomba <strong>manondro</strong> zavatra efa <strong>voalaza teo aloha</strong> (pronom, adverbe, etc.).<br><br>
      <strong>Règle :</strong><br>
      • Mampiasa <strong>pronom</strong> (il, elle, le, lui, etc.)<br>
      • Mampiasa <strong>adverbe</strong> (y, en, etc.)<br>
      • Manondro ny zavatra efa voalaza teo aloha<br>
      Ohatra : <em>J'ai un frère. <span class="color-anaphore">Il</span> est grand.</em>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay (Pronoms personnels)
      </div>
      <div class="exo-ligne">

      </div>
      <div class="exo-ligne">
        <span class="fr">Marie est belle. <span class="color-anaphore">Elle</span> est intelligente. (Tsara tarehy i Marie. <span class="color-anaphore">Manan-tsaina</span> izy.)</span>

      <div class="exo-ligne">

  <!-- 4. COMPARAISON ANAPHORE vs CATAPHORE -->
  <div class="card">
    <h2>4️⃣ Comparaison ANAPHORE vs CATAPHORE</h2>
    <div class="definition-mg">
      <strong><span class="color-anaphore">ANAPHORE</span></strong> → manondro zavatra efa <strong>voalaza teo aloha</strong><br>
      Ohatra : <em>J'ai un frère. <span class="color-anaphore">Il</span> est grand.</em><br><br>
      
      <strong><span class="color-cataphore">CATAPHORE</span></strong> → manondro zavatra izay <strong>holazaina any aoriana</strong><br>
      Ohatra : <em><span class="color-cataphore">Il</span> est grand, mon frère.</em><br><br>
      
      <strong>Fahasamihafana :</strong><br>
      • Ny <span class="color-anaphore">anaphore</span> dia <strong>miverina amin'ny</strong> zavatra efa voalaza (avy aloha).<br>
      • Ny <span class="color-cataphore">cataphore</span> dia <strong>manondro</strong> zavatra izay ho any aoriana (avy aoriana).<br>
      • Samy mampiasa <strong>pronom</strong> na <strong>adverbe</strong> izy roa.
    </div>
    <div class="exemple-modern" style="margin-top: 12px;">
      <div class="exo-titre">
        <span>🧪</span> Ohatra fampitahana mazava
      </div>
      <div class="exo-ligne">
        <span class="fr">ANAPHORE : Le chat dort. <span class="color-anaphore">Il</span> est fatigué. (Matoky ny saka. <span class="color-anaphore">Vizana</span> izy.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">CATAPHORE : <span class="color-cataphore">Il</span> est fatigué, le chat. (Vizana izy, ny saka.)</span>
      </div>
    </div>

  <!-- 5. AUTRES PROCÉDÉS DE REPRISE -->
  <div class="card">
    <h2>5️⃣ Autres procédés de reprise</h2>
    <div class="definition-mg">
      <strong>1. <span class="color-ellipse">Répétition avec ellipse :</span></strong><br>
      • Mamerina ny teny nefa mampihena ny fehezanteny.<br>
      Ohatra : <em>Pierre aime le café, Marie <span class="color-ellipse">aussi</span>.</em><br><br>
      
      <strong>2. <span class="color-anaphore">Pronom personnel :</span></strong><br>
      • Mampiasa pronom (il, elle, ils, elles, etc.)<br>
      Ohatra : <em>J'ai un frère. <span class="color-anaphore">Il</span> est grand.</em><br><br>
      
      <strong>3. <span class="color-anaphore">Pronom possessif :</span></strong><br>
      • Mampiasa pronom possessif (le mien, le tien, etc.)<br>
      Ohatra : <em>Mon livre est rouge, <span class="color-anaphore">le tien</span> est bleu.</em><br><br>
      
      <strong>4. <span class="color-anaphore">Pronom démonstratif :</span></strong><br>
      • Mampiasa pronom démonstratif (celui, celle, ceux, celles)<br>
      Ohatra : <em>J'aime cette robe, <span class="color-anaphore">celle</span> de Marie est belle.</em><br><br>
      
      <strong>5. <span class="color-anaphore">Adverbe :</span></strong><br>
      • Mampiasa adverbe (y, en, etc.)<br>
      Ohatra : <em>Je vais à Paris. J'<span class="color-anaphore">y</span> vais.</em>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
