  <!-- 1. COMMENT DIRE LA DATE -->
  <div class="card">
    <h2>1️⃣ Comment dire la date</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Mba hilazana ny <strong>date</strong>, dia ampiasaina ny <span class="color-jour">chiffre</span> + <span class="color-mois">mois</span> + <span class="color-annee">année</span>.<br><br>
      <strong>Structure :</strong><br>
      <em>le + <span class="color-jour">chiffre</span> + <span class="color-mois">mois</span> + <span class="color-annee">année</span></em><br><br>
      <strong>Exemples :</strong>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Nous sommes le <span class="color-jour">1er</span> <span class="color-mois">janvier</span> <span class="color-annee">2024</span>. (1er Janoary 2024 izao.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Aujourd'hui, c'est le <span class="color-jour">15</span> <span class="color-mois">juillet</span> <span class="color-annee">2024</span>. (15 Jolay 2024 androany.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Mon anniversaire est le <span class="color-jour">10</span> <span class="color-mois">mai</span> <span class="color-annee">2000</span>. (10 Mey 2000 ny fitsingerenan'ny andro nahaterahako.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Noël est le <span class="color-jour">25</span> <span class="color-mois">décembre</span>. (25 Desambra ny Noël.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">La rentrée est le <span class="color-jour">1er</span> <span class="color-mois">septembre</span> <span class="color-annee">2024</span>. (1er Septambra 2024 ny famerenana.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Ho an'ny <span class="color-jour">1er</span>, dia ampiasaina ny <strong>"premier"</strong> fa tsy "un".<br>
      Ohatra : <em>le 1er janvier</em> (fa tsy "le un janvier").
    </div>
  </div>

      Mba hilazana ny <strong>année</strong>, dia ampiasaina ny <strong>chiffres</strong> amin'ny teny frantsay.<br><br>
      <strong>Règle :</strong><br>
      • Ho an'ny taona <span class="color-annee">2000</span> sy mihoatra, dia ampiasaina ny <strong>chiffres complets</strong>.<br>
      Ohatra : <em>deux mille vingt-quatre</em> (2024)<br><br>
      • Ho an'ny taona <span class="color-annee">1900-1999</span>, dia azo ampiasaina ny <strong>chiffres séparés</strong>.<br>
      Ohatra : <em>dix-neuf cent quatre-vingt-dix</em> (1990)<br><br>
      <strong>Exemples :</strong>

  <div class="card">
    <h2>📌 Résumé des règles</h2>
    <div class="definition-mg">
      <strong>1. Structure de la date :</strong><br>
      • <em>le + <span class="color-jour">chiffre</span> + <span class="color-mois">mois</span> + <span class="color-annee">année</span></em><br>
      Ohatra : <em>le 10 janvier 2024</em><br><br>
      
      <strong>2. Avec le jour :</strong><br>
      • <em><span class="color-jour-semaine">jour</span> + le + <span class="color-jour">chiffre</span> + <span class="color-mois">mois</span> + <span class="color-annee">année</span></em><br>
      Ohatra : <em>lundi 10 janvier 2024</em><br><br>
      
      <strong>3. 1er :</strong><br>
      • <span class="color-jour">1er</span> → <strong>premier</strong><br>
      Ohatra : <em>le 1er janvier</em><br><br>
      
      <strong>4. Les jours et mois :</strong><br>
      • Misy <strong>majuscule</strong> amin'ny teny frantsay.<br>
      Ohatra : <em>Lundi, Janvier</em><br><br>
      
      <strong>5. Les années :</strong><br>
      • <span class="color-annee">2000+</span> → <em>deux mille</em> + chiffre<br>
      • <span class="color-annee">1900-1999</span> → <em>dix-neuf cent</em> + chiffre
