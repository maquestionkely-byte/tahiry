  <!-- Carte : Inona ny expression de conséquence ? -->
  <div class="card">
    <h2>🏷️ Inona ny "exprimer la conséquence" ?</h2>
    <div class="definition-mg">
      🔹 Ny <strong>conséquence</strong> dia mamaritra ny <strong>vokany</strong> (résultat) ateraky ny zavatra iray.<br><br>
      🔹 Mitovy amin'ny hoe <strong>"ka", "noho izany", "ary noho izany"</strong> amin'ny teny malagasy.<br><br>
      🔹 Misy fomba <strong>maro</strong> hilazana ny vokany :<br><br>
      <span class="icone">📌</span> <span class="color-donc">DONC</span> → ka / noho izany<br>
      Ohatra : <em>Il est malade, <span class="color-donc">donc</span> il reste à la maison.</em><br><br>
      <span class="icone">📌</span> <span class="color-ainsi">AINSI</span> → ary noho izany<br>
      Ohatra : <em>Il travaille bien, <span class="color-ainsi">ainsi</span> il réussit.</em><br><br>
      <span class="icone">📌</span> <span class="color-par-consequent">PAR CONSÉQUENT</span> → noho izany<br>
      Ohatra : <em>Il a menti, <span class="color-par-consequent">par conséquent</span> je ne lui fais pas confiance.</em><br><br>
      <span class="icone">📌</span> <span class="color-si-que">SI...QUE</span> → ... ka<br>
      Ohatra : <em>Il est <span class="color-si-que">si</span> gentil <span class="color-si-que">que</span> tout le monde l'aime.</em><br><br>
      <span class="icone">📌</span> <span class="color-tellement-que">TELLEMENT...QUE</span> → ... ka<br>
      Ohatra : <em>Il est <span class="color-tellement-que">tellement</span> fatigué <span class="color-tellement-que">qu'</span>il dort.</em>
    </div>
  </div>

  <!-- 1. DONC - Conséquence directe -->
  <div class="card">
    <h2>1️⃣ <span class="color-donc">DONC</span> (ka / noho izany)</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Ny <span class="color-donc">DONC</span> dia ampiasaina hilazana <strong>vokany</strong> (conséquence) mivantana. Apetraka eo anelanelan'ny fehezanteny roa.<br><br>
      <strong>Règle :</strong><br>
      • <span class="color-donc">DONC</span> + <strong>sujet + verbe</strong><br>
      • Apetraka <strong>aorian'ny</strong> fehezanteny voalohany<br>
      • Maneho vokany <strong>mivantana</strong>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Il est malade, <span class="color-donc">donc</span> il reste à la maison. (Marary izy, <span class="color-donc">ka</span> mijanona ao an-trano.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il a faim, <span class="color-donc">donc</span> il mange. (Noana izy, <span class="color-donc">ka</span> mihinana.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il pleut, <span class="color-donc">donc</span> nous restons à la maison. (Misy orana, <span class="color-donc">ka</span> mijanona ao an-trano isika.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elle a réussi, <span class="color-donc">donc</span> elle est contente. (Nahomby izy, <span class="color-donc">ka</span> faly.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Ny <span class="color-donc">DONC</span> dia ampiasaina hilazana <strong>vokany</strong> (conséquence) mivantana.
    </div>
  </div>

      • Apetraka <strong>aorian'ny</strong> fehezanteny voalohany<br>

      </div>
      <div class="exo-ligne">
        <span class="fr">Elle étudie beaucoup, <span class="color-ainsi">ainsi</span> elle a de bonnes notes. (Mianatra be izy, <span class="color-ainsi">ary noho izany</span> dia manana naoty tsara.)</span>

      • Ampiasaina amin'ny <strong>toetra</strong> (adjectif) na <strong>fomba</strong> (adverbe)<br>
      • Mitovy amin'ny <span class="color-si-que">SI...QUE</span>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Il est <span class="color-tellement-que">tellement</span> fatigué <span class="color-tellement-que">qu'</span>il dort. (Tena vizana izy <span class="color-tellement-que">ka</span> matory.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Elle est <span class="color-tellement-que">tellement</span> contente <span class="color-tellement-que">qu'</span>elle danse. (Tena faly izy <span class="color-tellement-que">ka</span> mandihy.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Il a <span class="color-tellement-que">tellement</span> faim <span class="color-tellement-que">qu'</span>il mange vite. (Tena noana izy <span class="color-tellement-que">ka</span> mihinana haingana.)</span>
      </div>

  <!-- 6. COMPARAISON SI...QUE vs TELLEMENT...QUE -->
  <div class="card">
    <h2>6️⃣ Comparaison SI...QUE vs TELLEMENT...QUE</h2>
    <div class="definition-mg">
      <strong><span class="color-si-que">SI...QUE</span></strong> → ...ka (amin'ny adjectif/adverbe)<br>
      Ohatra : <em>Il est <span class="color-si-que">si</span> gentil <span class="color-si-que">que</span> tout le monde l'aime.</em><br><br>
      
      <strong><span class="color-tellement-que">TELLEMENT...QUE</span></strong> → ...ka (amin'ny adjectif/adverbe)<br>
      Ohatra : <em>Il est <span class="color-tellement-que">tellement</span> fatigué <span class="color-tellement-que">qu'</span>il dort.</em><br><br>
      
      <strong>Fahasamihafana :</strong><br>
      • <span class="color-si-que">SI...QUE</span> → ampiasaina amin'ny <strong>adjectif</strong> sy <strong>adverbe</strong>.<br>
      • <span class="color-tellement-que">TELLEMENT...QUE</span> → ampiasaina amin'ny <strong>adjectif</strong>, <strong>adverbe</strong>, ary <strong>verbe</strong>.<br>
      • Samy maneho <strong>vokany</strong> avy amin'ny <strong>hamafiny</strong> (intensité).
    </div>
    <div class="exemple-modern" style="margin-top: 12px;">
      <div class="exo-titre">
        <span>🧪</span> Ohatra fampitahana mazava
      </div>

      </div>
      <div class="exo-ligne">
        <span class="fr">TELLEMENT...QUE : Il est <span class="color-tellement-que">tellement</span> gentil <span class="color-tellement-que">que</span> tout le monde l'aime. (Tena tsara fanahy izy ka tian'ny olona rehetra.)</span>
      </div>
    </div>
  </div>

  <!-- 7. RÉSUMÉ -->
  <div class="card">
    <h2>📌 Résumé des règles</h2>
    <div class="definition-mg">
      <strong>1. <span class="color-donc">DONC</span></strong> → ka / noho izany (vokany mivantana)<br>
      Ohatra : <em>Il est malade, donc il reste.</em><br><br>
      
      <strong>2. <span class="color-ainsi">AINSI</span></strong> → ary noho izany (vokany lojika)<br>
      Ohatra : <em>Il travaille bien, ainsi il réussit.</em><br><br>
      
      <strong>3. <span class="color-par-consequent">PAR CONSÉQUENT</span></strong> → noho izany (vokany ara-dalàna)<br>
      Ohatra : <em>Il a menti, par conséquent je ne lui fais pas confiance.</em><br><br>
      
      <strong>4. <span class="color-si-que">SI...QUE</span></strong> → ...ka (vokany avy amin'ny toetra/fomba)<br>
      Ohatra : <em>Il est si gentil que tout le monde l'aime.</em><br><br>
      
      <strong>5. <span class="color-tellement-que">TELLEMENT...QUE</span></strong> → ...ka (vokany avy amin'ny toetra/fomba/verbe)<br>
      Ohatra : <em>Il est tellement fatigué qu'il dort.</em>
