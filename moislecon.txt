  <!-- 4. COMMENT DIRE LA DATE -->
  <div class="card">
    <h2>4️⃣ Comment dire la date</h2>
    <div class="definition-mg">
      <strong>📖 Famaritana malagasy :</strong><br>
      Mba hilazana ny <strong>date</strong>, dia ampiasaina ny <strong>chiffre</strong> + <strong>mois</strong> + <strong>année</strong>.<br><br>
      <strong>Structure :</strong><br>
      <em>le + chiffre + mois + année</em><br><br>
      <strong>Exemples :</strong>
    </div>
    <div class="exemple-modern">
      <div class="exo-titre">
        <span>✨</span> Ohatra amin'ny teny frantsay
      </div>
      <div class="exo-ligne">
        <span class="fr">Nous sommes le <span class="color-printemps">1er</span> <span class="color-printemps">janvier</span>. (1er Janoary izao.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Aujourd'hui, c'est le <span class="color-ete">15</span> <span class="color-ete">juillet</span>. (15 Jolay androany.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Mon anniversaire est le <span class="color-printemps">10</span> <span class="color-printemps">mai</span>. (10 Mey ny fitsingerenan'ny andro nahaterahako.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">Noël est le <span class="color-hiver">25</span> <span class="color-hiver">décembre</span>. (25 Desambra ny Noël.)</span>
      </div>
      <div class="exo-ligne">
        <span class="fr">La rentrée est le <span class="color-printemps">1er</span> <span class="color-printemps">septembre</span>. (1er Septambra ny famerenana.)</span>
      </div>
    </div>
    <div class="highlight">
      💡 <strong>Tsarovy :</strong> Ho an'ny <strong>1er</strong>, dia ampiasaina ny <strong>"premier"</strong> fa tsy "un".<br>
      Ohatra : <em>le 1er janvier</em> (fa tsy "le un janvier").
    </div>
  </div>

  <!-- ==================== EXPRESSIONS AVEC LES MOIS ==================== -->
  <div class="card">
    <h2>📌 Expressions avec les mois</h2>
    <div class="definition-mg">
      <strong>Expressions courantes :</strong><br><br>
      • <strong>en janvier</strong> → amin'ny Janoary<br>
      • <strong>au mois de février</strong> → amin'ny volana Febroary<br>
      • <strong>en mars prochain</strong> → amin'ny Martsa ho avy<br>
